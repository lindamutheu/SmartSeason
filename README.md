# SmartSeason
SmartSeason is a web-based field monitoring system designed to help track crop progress across multiple fields during a growing season.
The system enables coordinators Admins and Field Agents to efficiently manage agricultural activities, monitor field status, and record crop updates in a structured and intuitive way.

Tech Stack

Frontend:HTML5, CCS3, & Vanilla Javascript

Backend:Django, Django Rest Framework

Databases:PostgreSQL
